FAQ

Pertanyaan Umum

Bagaimana cara membeli layanan?

Pilih layanan yang diinginkan, klik "Tambah ke Keranjang", lalu lanjutkan ke pembayaran. Pembayaran dapat dilakukan via QRIS atau Virtual Account melalui Pakasir.

Berapa lama proses aktivasi?

Panel Pterodactyl aktif dalam 5-15 menit setelah pembayaran. VPS aktif dalam 15-30 menit. Jasa IT tergantung tingkat kesulitan.

Apakah ada garansi?

Ya! VPS garansi 3 hari, Panel garansi 24 jam. Jika ada masalah teknis dari pihak kami, uang kembali 100%.

Bagaimana cara upgrade layanan?

Anda dapat upgrade kapan saja dengan membayar selisih harga. Hubungi admin via WhatsApp untuk proses upgrade.

Metode pembayaran apa saja yang tersedia?

Kami menerima pembayaran via QRIS (semua e-wallet: GoPay, OVO, Dana, ShopeePay, dll) dan semua m-banking. Scan QR code yang muncul saat checkout, pembayaran terdeteksi otomatis.

Apakah bisa request fitur custom untuk bot/website?

Tentu! Kami menerima request fitur custom. Hubungi admin via WhatsApp untuk diskusi kebutuhan dan estimasi harga. Harga menyesuaikan kompleksitas fitur.
